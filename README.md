**An active, production-ready fork of ember-validated-form**

This fork steps in to resolve critical deprecations and bugs, ensuring your forms remain highly stable in modern Ember environments. With a commitment to active maintenance and timely patches, it provides a reliable drop-in alternative to keep your application running smoothly without breaking changes.

**Quick install**

```bash
npm install git+https://github.com/Boundaryploice/ember-validated-form.git
```
